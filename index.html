Strict
ONOFF

The game bot will highlight a sequence of blocks, incrementing a step further each time. You just have to play it back correctly. Get 20 sequence correct to win the game. Strict mode ends the game if you input one incorrect sequence. But otherwise, the bot allows you to retry the failed sequence as many times as you want. Go kill some time!

Keybindings:

  • A to D --> game blocks
  • ? or / --> Help
  • x      --> close Help panel
  • s      --> toggle strict mode
  • Space  --> start / reset
